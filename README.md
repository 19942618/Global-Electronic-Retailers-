Excel Project

It feels so good to finally apply what I’ve been learning

Working on this project wasn’t easy — from merging data to fixing errors, I faced many challenges. But I’m proud to say I completed my first Excel project using Power Query and Power Pivot, analyzing the Global Electronic Retailer dataset.

Here are some of the insights I discovered:

2019 had the highest profit.

Male customers placed more orders.

Computers brought in the most revenue.

April recorded the lowest revenue.

North America had the biggest customer base.

In total, there were 26,326 orders, $55.8m in revenue, and $32.7m in profit.

This project really showed me how Power Query and Power Pivot can turn raw data into meaningful insights.

My Recommendations:

Repeat the winning strategies from 2019 to boost profit.

Create campaigns that engage more female customers.

Focus more on computers, since they bring in the most revenue.

Run promotions in April to increase sales.

Keep a strong base in North America, while also expanding to other regions.
